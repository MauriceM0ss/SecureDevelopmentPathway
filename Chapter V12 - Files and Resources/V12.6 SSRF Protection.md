# V12.6 SSRF Protection

## 12.6.1 Verify that the web or application server is configured with an allow list of resources or systems to which the server can send requests or load data/files from

De Server Side Request Forgery (SSRF) is vanuit de OWASP community toegevoegd als een van de meest impactvolle aanvallen, daarom dient hier ook zeker rekening mee te houden te worden. Bij een SSRF aanval laat een aanvaller de server verzoeken maken richting systemen, dit kunnen ook interne systemen zijn waardoor het mogelijk wordt voor een aanvaller om bijvoorbeeld firewalls te omzeilen. Om deze reden is het aanbevolen om een allow list te gebruiken die de server instrueert waarmee die mag verbinden.
